# v1 Prompt inicial
Eres un programador front end experto y debes realizar una página web, un index.html y un script.js en vanilla javascript, donde se tenga:
- Un campo de texto para recibir el string que se va a reversar.
- Un botón para activar el proceso de reversión. Llamalo "Reverse"
- Un campo de texto donde se tendrá la respuesta, que será la string al revés.
Ejemplos de texto reversado:
1. "AI4Devs" en la entrada se convierte en "sveD4IA" en la salida.

Hazme las preguntas necesarias antes de darme los dos entregables que sería un index.html y un script.js.
